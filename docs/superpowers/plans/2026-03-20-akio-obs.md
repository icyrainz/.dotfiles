# akio-obs Unified Observability Stack — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deploy a unified observability stack (VictoriaMetrics + VictoriaLogs + Grafana) on a dedicated LXC, then roll out Grafana Alloy agents to collect logs and metrics from all homelab LXCs and Proxmox hosts.

**Architecture:** New LXC 136 (akio-obs) runs three Docker containers. Alloy agents on each LXC ship Docker logs, systemd journal, and node metrics to the central stack. Grafana provides dashboards and log exploration.

**Tech Stack:** Proxmox (pct), Docker Compose, VictoriaMetrics, VictoriaLogs, Grafana, Grafana Alloy, NPM API, Pi-hole v6 API, Commando

**Spec:** `docs/superpowers/specs/2026-03-20-akio-obs-design.md`

**Credentials needed:** NPM creds (`NGINX_USERNAME` / `NGINX_PASSWORD`) and Pi-hole password (`PIHOLE_PASSWORD`) from fish extra (`~/.config/fish/include/ignore/extra.fish`).

- [ ] **Step 3: Write the Docker Alloy config**

This is `config-docker.alloy` — the variant for LXCs running Docker.

```bash
commando exec akio-lab/akio-obs 'cat > /etc/alloy/config.alloy << '"'"'ALLOCFG'"'"'
// --- Node Metrics ---
prometheus.exporter.unix "node" {}

prometheus.scrape "node" {
  targets    = prometheus.exporter.unix.node.targets
  forward_to = [prometheus.remote_write.victoriametrics.receiver]
  scrape_interval = "30s"
}

prometheus.remote_write "victoriametrics" {
  endpoint {
    url = "http://akio-obs:8428/api/v1/write"
  }
  external_labels = {
    host = constants.hostname,
  }
}

// --- Docker Log Discovery ---
discovery.docker "containers" {
  host = "unix:///var/run/docker.sock"
}

discovery.relabel "docker" {
  targets = discovery.docker.containers.targets
  rule {
    source_labels = ["__meta_docker_container_name"]
    target_label  = "container_name"
    regex         = "/(.*)"
  }
  rule {
    source_labels = ["__meta_docker_container_label_com_docker_compose_service"]
    target_label  = "compose_service"
  }
}

loki.source.docker "docker_logs" {
  host    = "unix:///var/run/docker.sock"
  targets = discovery.relabel.docker.output
  forward_to = [loki.process.add_host.receiver]
}

// --- Systemd Journal ---
loki.source.journal "journal" {
  forward_to = [loki.process.add_host.receiver]
  relabel_rules = loki.relabel.journal.rules
}

loki.relabel "journal" {
  forward_to = []
  rule {
    source_labels = ["__journal__systemd_unit"]
    target_label  = "unit"
  }
}

// --- Add host label + ship to VictoriaLogs ---
loki.process "add_host" {
  stage.static_labels {
    values = {
      host = constants.hostname,
    }
  }
  forward_to = [loki.write.victorialogs.receiver]
}

loki.write "victorialogs" {
  endpoint {
    url = "http://akio-obs:9428/insert/loki/api/v1/push"
  }
}
ALLOCFG'
```

Expected: Config file written.
